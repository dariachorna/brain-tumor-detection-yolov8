Brain tumor detection

Тренування моделі YOLOv8n для визначення трьох різновидів пухлин головного мозку. Використано 50 епох та стандартний розмір зображення 640.

Датасет містить 3 класи, а саме: glioma, meningioma, pituitary-tumor. Загальна кількість зображень 2540, з яких train - 1785, valid - 510, test - 254.

Проєкт складається з трьох файлів: код тренування та дві папки з прикладами вдалих і невдалих визначень.

Результати:
mAP50 = 0.73443
Precision = 0.79617
Recall = 0.72683

Датасет було обрано з двох причин: цікавість до медичної тематики та потенціалу CV у діагностиці та менша контрастність МРТ-зображень, що краще демонструє труднощі моделі.
Модель досягла хороших базових метрик, середнє значенн яких  0.75, вже на 10–15 епосі та добре навчилась розрізняти більшість низькоконтрастних пухлин. 
Основні помилками є неправильне визначення на дуже низькоконтрастних зображеннях, де пухлина плутається з іншими м'якими тканинами.
